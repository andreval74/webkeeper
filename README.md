# WebSuper — Landing Page

Landing page profissional para André W. Pereira (WebSuper).

## Estrutura

```
webkeeper/
├── index.html        ← página principal
├── assets/           ← imagens e recursos
├── .gitignore
└── README.md
```

## Como editar

1. Abra a pasta no VS Code
2. Edite o `index.html` diretamente
3. Abra no navegador para visualizar (ou use Live Server no VS Code)

## Deploy

```bash
git add .
git commit -m "descrição da mudança"
git push
```

GitHub Pages publica automaticamente em ~1 minuto.
